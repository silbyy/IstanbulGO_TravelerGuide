**ISTANBULGO - A DATA STRUCTURES PROJECT**

  We developed this project as a term project in our **Data Structures and Management** course so our main focus was demonstrating how        different data structures work together to solve real world problems

**1. Binary Search Tree (BST) - Area Management**

  We used a Generic BST to store and manage district (Area) data.
  We used this data structure to ensure efficient searching, insertion, and _alphabetical sorting_ of Istanbul's districts.
  Complexity: Provides an average of _O(logn) time complexity_ for search operations.

**2. Graph (Adjacency List) - The Transportation Network**

  Istanbul's transport system is modeled using a Weighted Graph.
  Nodes: Represent districts (Taksim, Kadikoy, etc.).
  Edges: Represent transport routes with weights representing travel time.
  Algorithm: We implemented _Dijkstra’s Algorithm_ to find the shortest path between any two points in the city.

**3. JSON & Linked Lists - Data Persistence**

  JSON: All location data (historical sites, cafes, etc.) is stored in a structured data.json file.
  Linked Lists: Used within the BST nodes to hold a dynamic list of locations for each specific district.
